# Ultrasonic Test

This folder contains firmware used to verify the operation of two HC-SR04 ultrasonic sensors connected to the ESP32.

## Objective

- Verify trigger and echo signals
- Measure obstacle distance
- Validate front and rear ultrasonic sensors
- Detect nearby obstacles during hardware bring-up

## Files

- **Ultrasonic_Test.ino** – Reads distance from two HC-SR04 ultrasonic sensors and reports measurements through the Serial Monitor.

## Hardware

- ESP32
- 2 × HC-SR04 Ultrasonic Sensors

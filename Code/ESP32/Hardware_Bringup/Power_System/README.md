# Power System Test

This folder contains firmware used to monitor the robotic dog's power system during hardware bring-up.

## Objective

- Measure battery voltage
- Measure battery current
- Calculate power consumption
- Detect low battery conditions
- Verify voltage and current sensor operation

## Files

- **Power_System.ino** – Reads battery voltage and current using analog sensors and displays power statistics through the Serial Monitor.

## Hardware

- ESP32
- 3S LiPo Battery
- Voltage Sensor Module
- ACS758 Current Sensor

# MPU6500 Test

This folder contains firmware used to verify communication with the MPU6500 Inertial Measurement Unit (IMU).

## Objective

- Verify I²C communication
- Read raw accelerometer values
- Read raw gyroscope values
- Read onboard temperature
- Validate sensor operation before implementing balance control

## Files

- **MPU6500_Test.ino** – Reads raw accelerometer, gyroscope, and temperature data from the MPU6500 and prints the values to the Serial Monitor.

## Hardware

- ESP32
- MPU6500 IMU

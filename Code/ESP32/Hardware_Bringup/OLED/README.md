# OLED Test

This folder contains firmware used to verify the operation of the SSD1306 OLED display connected to the ESP32.

## Objective

- Verify I²C communication
- Initialize the OLED display
- Display system status information
- Confirm correct display operation before integrating additional hardware

## Files

- **OLED_Test.ino** – Initializes the SSD1306 OLED and displays basic system information.

## Hardware

- ESP32
- SSD1306 OLED Display (128×64)
- I²C Interface

# FSR Test

This folder contains firmware used to test the Force Sensitive Resistor (FSR) sensors installed on the robotic dog's feet.

## Objective

- Verify sensor connectivity
- Read analog force values
- Detect foot-ground contact
- Validate sensor response before gait implementation

## Files

- **FSR_Test.ino** – Reads four FSR sensors and prints their values to the Serial Monitor.

## Hardware

- ESP32
- 4 × Force Sensitive Resistors (FSR)

## Project Objectives

The primary goal of this project is to develop an intelligent quadruped robot capable of:

- Stable locomotion
- Self-balancing
- Obstacle detection
- Terrain adaptation
- Autonomous navigation
- Modular hardware design
- AI-based decision making (future work)

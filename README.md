# 🐕 Robotic Dog

A 12-DOF quadruped robot currently under active development.

The goal of this project is to design and build a modular quadruped capable of stable walking, balancing, and autonomous navigation. The current prototype focuses on mechanical design, electronics integration, and hardware bring-up using an ESP32. Once the hardware and control software are validated, the project will migrate to an STM32-based controller for improved real-time performance.
